# Gestionnaire de bibliothèque

projet en **Rust** (TP1) : une application en ligne de commande pour gérer une collection de livres dans le terminal.

## Fonctionnalités

- Afficher tous les livres
- Ajouter un livre
- Rechercher un livre par titre
- Modifier le statut d'un livre (Disponible / Emprunté)
- Afficher des statistiques (nombre de livres, total et moyenne de pages, disponibles, empruntés)
- Quitter

## Lancer le projet

```bash
cargo run
```
